# Варіанти зображень для блоку «З чим працюємо»

Підібрано **4 напрями** (кожен — з безкоштовних стоків). Поточні URL у коді — у `category_fallback.py` (Unsplash). Нижче — **альтернативи**, якщо захочеш інший настрій тієї ж тематики.

**Палітра сайту:** `#0A1628`, `#2563EB`, `#EEF5FF`, білий — фото краще брати з холодним світлом, небом, металом, склом; теплі кадри лишають під легкий градієнт у [`carousel.css`](static/landing/css/carousel.css) (оверлей на `.category-card-image`).
